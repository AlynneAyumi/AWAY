
# Projeto Integrador: Sistema de Gestão de Processos para o Patronato de Foz do Iguaçu

## Descrição do Projeto

Este projeto tem como objetivo desenvolver um **sistema de gestão de processos** para apoiar o Patronato a ter um controle ampliado dos processos e presidiários que comparecem ao local. A plataforma visa melhorar a eficiência na administração das atividades, no controle dos usuários dos serviços e na gestão dos recursos disponíveis, promovendo um impacto positivo na comunidade atendida.

## Funcionalidades Principais

- **Cadastro de Usuários:** Registro de pessoas atendidas, incluindo informações como perfil socioeconômico e necessidades específicas.
- **Relatórios e Estatísticas:** Geração de relatórios sobre os atendimentos realizados, evolução dos usuários e utilização de recursos.
- **Controle de Presença:** Histórico de atendimentos e controle da situação de cada individuo.

- Node.js (versão 14.0 ou superior)

## Como Executar Localmente

### 1. Clone o Repositório
```bash
git clone https://github.com/username/projeto-patronato.git
```

### 2. Instale as Dependências
No diretório do backend e frontend, execute o comando:
```bash
npm install
```

### 3. Inicie o Servidor Backend
```bash
cd backend
npm start
```

### 4. Inicie o Servidor Frontend
```bash
cd frontend
npm start
```

### 5. Acesse a Aplicação
Abra o navegador e acesse `http://localhost:3000` para visualizar a interface do sistema.
